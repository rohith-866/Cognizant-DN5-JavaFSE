# JUnit Demo - Exercise 1

A simple Java Maven project demonstrating JUnit 4 unit testing.

## Project Structure

```
JUnitDemo/
├── pom.xml
├── src/
│   ├── main/java/com/example/Calculator.java
│   └── test/java/com/example/CalculatorTest.java
```

## How to Run Tests

```bash
mvn test
```

## Dependencies

- JUnit 4.13.2
